SPA for work with local database and with following abilities:
- Login/register user
- Change user's information
- Add and remove user's notes (aka todo list)

## Install

```sh
yarn
```

in other terminal

```sh
cd server/
yarn
```

## Usage

```sh
yarn start
```

in other terminal

```sh
cd server/
yarn dev
```
